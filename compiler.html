cloudGENERAR

Esta especie de "editor" permite crear archivos con los estilos y la forma de las paginas de esta app. Existen varios comandos basicos para crear parrafos con texto y tarjetas con información links.

¿Comó se usa?
Hay que poner todo el codigo en el campo de texto de arriba, le das a GENERAR (también funciona con ctr+s). Te va a aparecer el resultado y abajo el codigo html que luego se sube a la app y recién ahí ya sería visible para todos en la app. 



           
Estructura de tarjeta

           

Nueva tarjeta -new_card
__  Division para imagen -img_div
__  __  Imagen -imagen NOMBREARCHIVO o http:LINK_EXTERNO //tambien se puede colocar un video con -video https://www.youtube.com/embed/thzm6oqK8O4
__  __  Posible titulo sobre la imagen -titulo
__  Cerrar img div -close_img_div
__  Division para el contenido -content
__  __ parrafo -p
__  __  __  Cualquier texto
__  __ cerrar_parrafo -p
__  Cerrar division contenido -close_content
__  Division LINKS -card_action
__Cerrar division LINKS -close_action
CERRAR TARJETA -close_card


para crear enlaces que te lleven a paginas dentro de la app en la parte donde se pone el link se pone: #!NombreDeLaSeccionOPagina
ej:


-card_action

-new_link #!home
-texto_simple Inicio
-close_link

-close_action
